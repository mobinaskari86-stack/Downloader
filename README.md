# 🎬 ربات دانلودر یوتیوب

## قابلیت‌ها
- دانلود ویدیو با کیفیت 360p و 720p
- دانلود فقط صدا به فرمت MP3
- نمایش اطلاعات ویدیو قبل از دانلود

## مراحل دیپلوی روی Railway

### 1. ساخت ربات تلگرام
- به @BotFather برو
- /newbot بزن
- اسم و یوزرنیم بده
- توکن رو کپی کن

### 2. آپلود روی Railway
- پروژه جدید بساز
- فایل‌ها رو آپلود کن

### 3. تنظیم متغیر محیطی
در بخش Variables این رو اضافه کن:
```
BOT_TOKEN = توکن_ربات_تلگرامت
```

### 4. دیپلوی
Railway خودکار دیپلوی میکنه ✅

## نکته مهم
- حداکثر حجم فایل: 50MB (محدودیت تلگرام)
- بعد از دیپلوی، یوزرنیم ربات رو توی bot.py جایگزین @YourBotUsername کن
